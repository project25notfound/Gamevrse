ColorRush

Memory • Speed • Competition

Game Rules

  • Watch the color & sound sequence carefully
  • Remember the pattern - it gets longer each round
  • Repeat by clicking buttons in correct order (or press keys 1–4)
  • You have 30 seconds each round to submit your answer
  • Start with 2 lives - lose one for each mistake or timeout

Power-Ups (if enabled)

  • Second Chance - Replay the sequence once per game after a wrong answer
  • Freeze - Pause your timer for 3 seconds (use once per round)
  • Pattern Peek - Reveal the sequence during play phase (use once per game)

Game End Scenarios

  • Normal Victory: Last player standing or highest score after max rounds
  • DRAW (3+ players): If all remaining players lose their last life in the SAME round, the game ends in a draw - no winner!
  • 1v1 SUDDEN DEATH: When exactly 2 players both lose their last life in the same round, they enter sudden death instead of a draw

Sudden Death Rules (1v1 Only)

  • When: Triggers ONLY when exactly 2 players both fail in the same round
  • Revival: Both players are revived with 1 life for a final showdown
  • Challenge: Fixed 5-tile sequence - first to get it right wins
  • If both fail: Sudden death continues with a new round (NOT a draw)
  • Power-ups: Still work in sudden death (timer pauses during Second Chance)

Elimination & Spectating

  • Eliminated when you lose your last life
  • Watch the match progress and see the live scoreboard
  • Eliminated players cannot rejoin (except in 1v1 sudden death)
  • See who's still alive and track the current round

Quick Tips

  • Use keyboard shortcuts (1-4) for faster input
  • Save your Second Chance for difficult rounds
  • Use Freeze when you need more time to think
  • 3+ players all fail = DRAW | 2 players both fail = SUDDEN DEATH
